== Description ==

Accept Visa, MasterCard, and more directly on your store with the Vapulus gateway for WooCommerce, including Payments for mobile and desktop.

= Why choose Vapulus? =

Setting up VAPULUS and starting to receive your payments is completely free, VAPULUS gateway is very transparent with its fees. Transactional fees are deducted per successful transaction! You get paid once you have fully completed your business profile and reached the payout limits. Or the payout time limits.

VAPULUS also supports the subscriptions and reoccurring payments. When a customer pays your business, they are set up in VAPULUS as a customer, and listed in your customers database, please check your VAPULUS portal. And since we are PCI compliant for security once your user is resisted and revisit your business to buy again, they can check out using the same card using a username only. Cutting check time to 1 tenth of the initial purchase.

CUSTOMER ACQUISITION & GROWING YOUR BUSINESS

= Integration =

Vapulus gateway requires a [vapulus merchant account](https://app.vapulus.com/business/).
This plugin will redirect you to make a vapulus merchant account,
Then create for your current wordpress a website ID inside vapulus dashboard
Than automatically add this website ID inside vapulus gateway plugin settings in your blog
This will allow you to accept payments online.

== Installation ==

Please note, v1 of vapulus gateway requires WooCommerce 2.6 or above.


= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “WooCommerce Vapulus Payment Gateway” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
